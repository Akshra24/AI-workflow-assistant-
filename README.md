An AI-powered workflow automation tool that converts unstructured text or documents into structured, actionable outputs such as summaries, key points, and action items.

---

## 🚀 Features

* 📄 Accepts **text and PDF inputs**
* 📝 Generates **concise summaries**
* 📌 Extracts **key points**
* ✅ Identifies **actionable tasks**
* ⚡ Fast and lightweight (no API required)
* 💻 Simple and interactive UI using Streamlit

---

## 🧠 How It Works

The system follows a structured pipeline:

1. **Input Processing**

   * Accepts raw text or extracts content from PDFs

2. **Preprocessing**

   * Cleans and segments text into sentences

3. **Processing Logic**

   * Generates summary (first key sentences)
   * Extracts key points
   * Detects action items using rule-based filtering

4. **Output Formatting**

   * Displays results in a structured and readable format
